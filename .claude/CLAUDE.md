# swana-olympics-invite

Single-page HTML invitation for the SWANA Olympics event.

## Stack
- Plain HTML/CSS/JS — no build step
- Google Fonts: Bebas Neue (title), Nunito (details)
- Web Audio: local MP3 (K'NAAN - Wavin' Flag), starts at 12s
- Canvas confetti (vanilla JS)

## Event Details
- Date: April 23, 2026
- Time: 4 PM – 7 PM
- Location: The Lawn
- Org: SWANA student org

## Files
- `index.html` — the whole page, self-contained
- `K'NAAN Wavin Flag World Cup Song.mp3` — audio file (starts at 12s to skip quiet intro)

## Hosting
Deploy as a static site (Vercel, GitHub Pages, Netlify).
QR code on physical business card points to hosted URL.
